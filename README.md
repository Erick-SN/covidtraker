# COVID-19 TRAKER
Application to observe the behavior of COVID-19 in the world
You can observe the cases of contagion, deaths and recovery by city and worldwide.

## Starting 🚀

You just need to clone the project

### Pre requirements 📋

_This project uses the package manager npm and was bootstrapped with [Create React App](https://github.com/facebook/create-react-app)._  
**So you should have nodejs, npm and npx installed to be able to run the project**

### Installation 🔧

_To install project dependencies:_

```bash
npm install
```

## Run ⚙️

```bash
npm run start
```

## Built with 🛠️

- **[ReactJS]** - Framework
- **[NPM]** - Package manager
- **[MaterialUI]** - Used to generate styles
- **[ChartJS]** - Work with charts in react
- **[Leaflet]** - Alternative to google maps
